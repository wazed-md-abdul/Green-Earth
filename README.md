
# 🌱 Green Earth

<p align="center">
A responsive environmental awareness website that promotes sustainability and eco-friendly living.
</p>

---

## 🌍 About

Green Earth is a modern and responsive website created to spread awareness about protecting the environment and adopting sustainable habits. The project focuses on clean design, responsive layouts, and interactive UI elements built with modern frontend technologies.

---

## 🚀 Features

- 🌿 Clean and modern UI
- 📱 Responsive design for all devices
- ⚡ Interactive functionality using JavaScript
- 🎨 Styled using Tailwind CSS
- 🌎 Environmental awareness theme
